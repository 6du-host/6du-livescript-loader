# vue-livescript-loader

## Usage

``` javascript
{
    module: {
        loaders: [
            { test: /\.vue$/, loader: "vue-livescript-loader" },
            { test: /\.ls$/, loader: "vue-livescript-loader" }
        ]
    }
}
```
